# Customer-Shopping-Behavior-Analysis
Customer Shopping Behavior Analysis using Python, MySQL, and Power BI. This project explores customer purchasing patterns, spending habits, product performance, discount impact, subscription behavior, and revenue trends through data cleaning, SQL analysis, and interactive dashboards.

Project Overview

This project analyzes customer shopping behavior to uncover valuable business insights that can help improve sales, customer retention, and marketing strategies. The analysis combines Python for data preprocessing, MySQL for advanced querying, and Power BI for interactive dashboard creation.

Objectives
Analyze customer purchasing patterns
Compare spending behavior across customer segments
Evaluate product performance using ratings and sales
Measure the impact of discounts on purchases
Study subscription and loyalty behavior
Identify high-value customers
Generate business recommendations based on data
